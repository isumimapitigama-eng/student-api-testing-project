# Student Management API

## Overview
A RESTful Student Management API developed using Python. The API allows users to manage student records through CRUD operations.

## Features
- Create Student (POST)
- Retrieve Student Data (GET)
- Update Student Information (PUT)
- Delete Student Records (DELETE)

## Testing
The API was tested using Postman to verify:
- Endpoint functionality
- Response status codes
- Data validation
- Error handling

## Technologies Used
- Python
- Flask
- REST API
- Postman
- JSON
- Visual Studio Code
